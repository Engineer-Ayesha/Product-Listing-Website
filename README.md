# Product Listing Website

A responsive **Product Listing Website** built using **HTML, CSS, and JavaScript**.  
Features:

- Display product title, owner, price, and description
- Next / Previous navigation with circular loop
- Fade animation on product change
- Fully responsive design (desktop, tablet, mobile)
- Images used from Unsplash & local folder

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- GitHub Pages for live demo
